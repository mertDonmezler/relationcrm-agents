# Quick Start Guide

## 1. Prerequisites

- Claude Code installed
- Flutter SDK (3.x)
- Firebase CLI
- Node.js 18+

## 2. Install Plugins

```bash
# Add marketplace
/plugin marketplace add ./RelationCRM-Agents

# Install all plugins
/plugin install flutter-mobile-dev backend-api-dev ai-relationship-engine integrations-hub quality-assurance infrastructure-ops growth-monetization privacy-compliance orchestration-workflows
```

## 3. Create Project

```bash
# Scaffold Flutter app with clean architecture
/scaffold-app
```

## 4. Build Your First Feature

```bash
# Authentication with Google Sign-In
/full-feature "user authentication with Google"
```

## 5. Key Workflows

### Feature Development
```bash
/full-feature "contact management"
```

### Sprint Planning
```bash
/sprint-plan
```

### Release
```bash
/release "1.0.0"
```

## 6. Agent Quick Reference

| Need | Agent | Command |
|------|-------|---------|
| Flutter code | flutter-architect | "implement X screen" |
| UI design | ui-designer | "design X component" |
| API endpoint | api-developer | "create X endpoint" |
| AI feature | ai-architect | "add AI for X" |
| Tests | test-architect | "write tests for X" |
| Security | security-auditor | "audit X" |
| Deploy | devops-engineer | "deploy to production" |

## 7. Tips

1. **Start with orchestrator**: Use `/full-feature` for complex features
2. **Privacy first**: Always consult privacy-architect for data flows
3. **iOS 18**: Check ios18-contacts skill before contact features
4. **AI costs**: Use haiku for 80% of AI requests
